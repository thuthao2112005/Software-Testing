# Software Testing for Login & Registration System

## Overview
This project focuses on manual and automated testing for the Login and Registration modules of a web application to verify functionality and improve software quality.

## Responsibilities
- Designed test cases for Login and Registration features.
- Performed manual testing and automated testing using Selenium WebDriver.
- Applied software testing techniques such as Equivalence Partitioning, Boundary Value Analysis, Decision Table, and State Transition Testing.
- Recorded and analyzed test execution results.

## Results
- Designed and executed **62 test cases**.
- Automated **56 test cases** using Selenium.
- Achieved comprehensive test coverage for Login and Registration workflows.
- Identified defects and validated system behavior against functional requirements.

## Technologies
Python • Selenium WebDriver • ChromeDriver • Microsoft Excel
